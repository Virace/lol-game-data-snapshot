# lol-game-data-snapshot

Current League of Legends LCU JSON snapshot generated from Riot client manifests.

`main` is the latest snapshot. Use Git commit or tag references for historical reads.

## Raw URLs

```text
https://raw.githubusercontent.com/Virace/lol-game-data-snapshot/main/global/default/v1/champion-summary.json
https://raw.githubusercontent.com/Virace/lol-game-data-snapshot/main/global/default/v1/maps.json
https://raw.githubusercontent.com/Virace/lol-game-data-snapshot/main/global/default/v1/items.json
https://raw.githubusercontent.com/Virace/lol-game-data-snapshot/main/global/default/v1/summoner-spells.json
https://raw.githubusercontent.com/Virace/lol-game-data-snapshot/main/global/default/v1/perks.json
https://raw.githubusercontent.com/Virace/lol-game-data-snapshot/main/global/default/v1/skins.json
```

The repository stores JSON only. Audio files, image files, WAD files, and GAME resources are not committed.
